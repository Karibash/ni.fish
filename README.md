# ni.fish

Alternative [ni](https://github.com/antfu/ni) written in fish.

## 🔧 Supports

- npm
- yarn
- pnpm
- bun

## 🚀 Installation

Install with [Fisher](https://github.com/jorgebucaran/fisher):

```
$ fisher install Karibash/ni.fish
```
